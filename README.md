# Steam Machine Hardware Reference

Early technical documentation collected from a retail Steam Machine.

This repository contains hardware information gathered using standard Linux tools, including:

- SteamOS version
- CPU information
- Memory configuration
- PCI devices
- BIOS information
- Graphics stack
- Vulkan implementation
- Storage configuration

The goal is to provide verified reference material for the community.
